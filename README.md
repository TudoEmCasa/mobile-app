# tudo_em_casa

Offline-first Flutter mobile app for household inventory management.

## Getting Started

Use Mise for project commands:

```bash
flutter pub get
flutter analyze
flutter test
```

Environment details and troubleshooting are documented in `docs/environment.md`.
